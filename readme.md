# Discord Dynamico bot

### Installation
- Clone this repository
```shell
git clone https://github.com/CoffeBuffet/dynamico.git
```
- Enter the directory
```shell
cd dynamico
```
- Install dependencies
```shell
yarn install
```
- Edit .env according to your configuration, & push db
```shell
prisma db push
```
- Run the bot
```shell
yarn start
```
